# OfficeMate 手工测试题

这份题库基于当前示例制度文档整理，可直接用于课堂演示、功能验收和答辩测试。

建议使用方式：

1. 先在左侧切换到对应分类，再提问。
2. 观察回答是否包含正确规则、步骤和引用来源。
3. 重点看系统是否能在材料不足时明确提示“未找到明确依据”。
